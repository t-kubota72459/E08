# 任意の論理回路の構成法

好きな回路（任意の入出力）をどうやったら機械的に作れるか、ということを説明する。

ここでは、３入力の多数決回路を取り上げる。

## 回路（動作）仕様

３人の審査員 (A, B, C) がボタンで投票するシステムを作る。**２人以上が賛成 (ボタンを押す=1)** のときだけ、合格ランプ (Y) が点灯 (1) する回路を設計せよ。

この回路は３つの入力があり、出力は１つということになる。

## ステップ1. 真理値表

回路を作るファーストステップは真理値表を作ることからはじまる。

|入力A|入力B|入力C|出力Y|
|----|----|----|----|
|    |    | 0  |    |
|    |    | 1  |    |
|    |    | 0  |    |
|    |    | 1  |    |
|    |    | 0  |    |
|    |    | 1  |    |
|    |    | 0  |    |
|    |    | 1  |    |

## ステップ2. 出力 1 に注目

「出力 Y が 1 になる」行だけを抜き出す。

- `A=0, B=1, C=1` のとき
- `A=1, B=0, C=1` のとき
- `A=1, B=1, C=0` のとき
- `A=1, B=1, C=1` のとき

## ステップ3. AND と OR で結ぶ

このどれか一つでも満たされればいい、ということは論理式で

$Y = (\overline{A}\cdot B \cdot C) + (A \cdot \overline{B} \cdot C) + (A \cdot B \cdot \overline{C}) + (A \cdot B \cdot C)$

と表現できる。

**AND (積) で条件をまとめ、それらを OR (和) で繋げば任意の回路を作ることができる。** このような構成を **AND-OR 構成法、加法形** という。また、すべての項に入力全部があるとき、**積和標準形**という。
